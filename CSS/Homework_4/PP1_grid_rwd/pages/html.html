HTML Basics

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

For starters we need to have a simple web application. To do that we can either do it by hand or use Emmet Abbreviation to fill it in for us (using IDE like VS CODE by typing ! or !!!):

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0"& gt;
<title>Document</title>
</head>
<body>
<p>
</p>
</body>
</html>

Explanation of basic tags:

  • The <!DOCTYPE html> declaration defines that this document is an HTML5 document.
  • The <html> element is the root element of an HTML page.
  • The <head> element contains meta information about the HTML page.
  • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab).
  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The <h1> to <h6> elements define heading from largest-h1 to smallest-h6.
  • The <p> element defines a paragraph for text, lists, etc.
  • The <br> element defines page break (new line).
  • The <hr> element defines horizontal line.

We can include pictures using example code below:

<img src="path to image file or link to image (if hosted online), has to end with image type (jpeg, jpg, png, etc)" alt="text version of picture in case it does not load"/>
I.E. <img src="https://i.imgur.com/Blzt3qG.png" alt="Programmer next to pc"/>

We can also use unordered / ordered lists:

Unordered list example:

  • First
  • Second
  • Third
  • Fourth

Code:

<ul>
<li>First</li>
<li>Second</li>
<li>Third</li>
<li>Fourth</li>
</ul>

Ordered list example:

  1. First
  2. Second
  3. Third
  4. Fourth

Code:

<ol>
<li>First</li>
<li>Second</li>
<li>Third</li>
<li>Fourth</li>
</ol>

Tags used for list:

  • The <ul> start of unordered list.
  • The <ol> start of ordered list.
  • The <li> states each list item, must be in between <ul> or <ol> tags.

Example of table (how it looks like on a website):

Number Name Surname
1 Jon Doe
2 Joa Doe

Tags used for table:

  • The <table> defines table tag.
  • The <th> defines table head (top of table, name of lists).
  • The <tr> defines table row.
  • The <td> defines table data (in this case numbers,names, surnames).
